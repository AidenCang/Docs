# Material for MkDocs

A Material Design theme for [MkDocs][1].

[![Material for MkDocs](docs/assets/images/material.png)][2]

  [1]: https://www.mkdocs.org
  [2]: https://squidfunk.github.io/mkdocs-material/

## 环境设置

使用`pip`安装最新版本的软件:

``` sh
pip install mkdocs-material
```

配置文件路径`mkdocs.yml`
